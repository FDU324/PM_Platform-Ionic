## Initial

> npm install -g ionic cordova

> git clone https://github.com/FDU324/PM_Platform-Ionic.git pm-platform

> cd pm-platform

> npm install

### Browser

> ionic serve

### Android

> ionic cordova platform add android

> ionic cordova run android --device

#### Production Builds

> ionic cordova run android --prod --release
